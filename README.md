# Projeto Pratico PHP

Este projeto implementa uma agenda de contatos com as seguintes funcionalidades: 
 - Autenticação por Login e Senha
 - Listagem de contatos
 - Alteração dos dados dos contatos
 - Remover contatos
 - Cadastro de usuário
 
Tecnologias ultilizadas:
 - PHP
 - MySQL
 - Apache
 - Twitter Bootstrap
